Project Overview

Code-to-Graph is a comprehensive three-stage self-learning project that transforms complex mathematical concepts into an accessible, interactive web application. Built for the Matrices & Differential Calculus curriculum at Shah and Anchor Kutchhi Engineering College.

Stage 1: Theory

Deep dive into graph theory fundamentals, adjacency matrices, and Hamming(7,4) error correction codes. Mathematical foundations that power real-world systems.

Stage 2: Implementation

Python-based algorithm development with comprehensive testing. 74 unit tests ensuring 100% correctness before deployment.

Stage 3: Web Application

Client-side JavaScript implementation with Canvas-based visualization. Zero backend required, instant global deployment.

Key Features

Graph Network Visualizer

  • Real-time network topology visualization using HTML5 Canvas
  • Automatic adjacency matrix generation
  • Node degree calculation and hub identification
  • Network density and connectivity metrics
  • Support for custom node names and edge definitions

Hamming(7,4) Error Correction

  • 4-bit to 7-bit message encoding with parity bits
  • Automatic syndrome calculation
  • Single-bit error detection and correction
  • Visual representation of parity bit positions
  • Interactive error simulation for learning
